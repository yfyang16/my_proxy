## A simple multi-thread proxy
### Author: Yufeng Yang

The project was inspired by the proxy lab of cmu 15213 and there are still a
lot of to-do things.

### Run
```
cd my_proxy
make all
./proxy <available port>
```

### Key Files
```
proxy.c
- sbuf.c
- mycache.c
- list.h
- csapp.c
```
